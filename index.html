Wat is Elasticsearch?

  • Document oriented search engine
    JSON based, Apache Lucene
  • Schema free
  • Distributed
    Scales Up + Out, Highly Available
  • Multitenancy
    Dynamisch aanmaken en verwijderen van indices
  • API centric & RESTful
